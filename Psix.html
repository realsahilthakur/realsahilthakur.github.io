Attendance Management

Built an Attendance Management System using a Fingerprint Sensor and Arduino with Arduino IDE. This project was created using knowledge of Arduino, Fingerprint, and Arduino IDE. Users can store their fingerprint, and every time they rescan it, the time and date will be saved. It can save 200+ fingerprints. It automatically removes stored time and date after 15 days from the system. Whole data will be saved in the Arduino memory itself.
