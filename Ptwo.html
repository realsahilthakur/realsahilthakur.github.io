Smart Waste Classification System

Waste Classifier Using CVzone


This is a waste classifier program where users can show waste to the camera, and with the help of CVzone, it will automatically detect the type of waste, which can be one of the following four: Recycle waste, Hazardous waste, Food waste, or Residual waste.

Additionally, if a user is in front of the camera, it displays a message on the screen saying "Human Detected!".

The Smart Waste Classification System represents a significant step towards automating and enhancing waste management processes using computer vision and AI.

By providing real-time feedback and guidance, it can play a crucial role in improving recycling practices and contributing to environmental sustainability. While the system has some limitations, future enhancements and integrations offer promising opportunities for broader application and impact.



A diagram of the Waste Classifier system.


Key Features

  • Real-Time Waste Detection: Uses CVzone to identify waste types (Recycle, Hazardous, Food) via camera input.
  • Human Detection: Displays "Human Detected!" message when a person is in front of the camera.
  • Automated Classification: Leverages computer vision and AI for instant waste type recognition.
  • Environmental Impact: Encourages recycling and proper waste disposal through real-time feedback.
